== Description ==
This plugin allows you to display a feed of your Rougeo trails on any page or post of your WordPress website.

A Rougeo account is required in order to use the plugin. Your account Publisher ID is then used to show a feed of your trails.

== Installation ==
1) After installing, go to Settings > Rougeo Connect.
2) Copy the shortcode displayed in the blue box. 
3) Paste the shortcode into any page on your website, and customise the \'publisherid\' and \'lang\' options.

== Frequently Asked Questions ==
Is there a limit to how many trails I can show?
No there are no limits.

Can I show more than one feed on my website?
Sure. If you have more that one publisher ID then you are free to use as many as you want.
